MY PROJECT 

                                                             The project that I performed in my college is "IOT BASED PHOTO-VOLTAIC MONITORING SYSTEM USING NODEMCU ESP2866".

                It is an hardware project which consist of Arduino,NODEMCU ESP2866,battery,solar panel,lcd display,power supply,volatege regulators,temparature sensors,volatge sensor,
                current sensor,motor and an cloud server for reading to display through wifi module.


ABSTRACT

This project aims to develop a measurement of solar energy using Arduino Board technology. In this project, we are measuring the parameters like voltage ,current and power. These parameters are the input values for the Arduino and the output displays at the Liquid Crystal Display (LCD) screen.The LCD screen displays output of voltage, power and current value. The purpose of Arduino is to convert the analog input of the parameters to the digital output and displays via LCD screen. These data will be uploaded to server through NodeMCU.

EXISTING METHOD

In past we don't have any automatic technologies for rotating the solar panel. It was not provide sufficient output continuously. It is difficult to rotate solar panel every time on the day manually at plant. So power is not sufficient for all the time. Traditionally cleaning system was done manually.
